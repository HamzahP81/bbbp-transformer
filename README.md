# BBBP Transformer Classifier

## Overview
This project explores and attempts to predict the blood-brain barrier penetration value of various drugs using machine learning and complex transformer-based models. We'll be using the [BBBP Dataset] (https://deepchem.readthedocs.io/en/latest/api_reference/moleculenet.html#bbbp-datasets).

## Installation
